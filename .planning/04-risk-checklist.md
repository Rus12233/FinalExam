## 위험: Hive TypeAdapter 스키마 변경 시 데이터 호환성 깨짐

- 카테고리: 기술
- 발생 가능성: 중
- 영향도: 상
- 대응:
  - 예방: `DiaryEntry` 필드 추가 시 반드시 기본값 제공, TypeAdapter의 `typeId` 변경 금지, 변경 전 기존 데이터 백업 스크립트 준비
  - 발생 시: Hive box를 초기화하고 재생성, 마이그레이션 로직을 `main.dart` 초기화 단계에 추가
- 모니터링 지표: `DiaryEntry` 모델 파일 변경 커밋마다 실기기 데이터 유지 여부 수동 확인

## 위험: 파티클 애니메이션 성능 저하 (구형 기기 / 웹)

- 카테고리: 기술
- 발생 가능성: 중
- 영향도: 중
- 대응:
  - 예방: 파티클 개수를 조절 가능한 상수로 선언, 웹 빌드 시 `--web-renderer canvaskit` 옵션 적용
  - 발생 시: 파티클 수 축소(30개 → 15개), 애니메이션 duration 단축으로 폴백
- 모니터링 지표: Flutter DevTools Performance 탭에서 frame rate 16ms 초과 여부 확인

## 위험: 웹 빌드 base-href 설정 오류로 배포 실패

- 카테고리: 기술
- 발생 가능성: 중
- 영향도: 중
- 대응:
  - 예방: `flutter build web --release --base-href "/app/"` 명령어를 `docs/deploy.md`에 명시, CI 빌드 스크립트 작성
  - 발생 시: `index.html`의 `<base href>` 태그를 배포 경로에 맞게 수동 수정
- 모니터링 지표: 배포 후 URL 직접 접속 및 새로고침 정상 동작 확인

- 카테고리: AI 의존

- 영향도: 상
- 대응:

- 모니터링 지표: 주요 파일(`diary_provider.dart`, `plant_state.dart`, `plant_widget.dart`)을 손으로 한 번씩 재작성해 보기

- 영향도: 중
- 대응:
