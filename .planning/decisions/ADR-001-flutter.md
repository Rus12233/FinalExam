# ADR-001: Flutter 선택

## 상태
채택됨

## 맥락
iOS와 Android 양쪽을 지원하는 감성 다이어리 앱이 필요했습니다.
파티클 애니메이션, 부드러운 식물 성장 연출이 핵심 기능이므로 UI 성능이 중요했습니다.

## 결정
Flutter(Dart)를 사용합니다.

## 이유
- **단일 코드베이스**: iOS/Android/Web을 하나의 코드로 커버
- **60fps 렌더링**: Skia/Impeller 엔진 기반으로 애니메이션이 부드러움
- `flutter_animate` 패키지로 파티클·성장 효과를 선언형으로 구현 가능
- React Native 대비 네이티브 UI 의존도가 낮아 감성 커스텀 디자인에 유리

## 트레이드오프
- Dart 언어를 별도로 학습해야 함
- 앱 바이너리 크기가 네이티브 대비 큼 (~8MB 추가)
