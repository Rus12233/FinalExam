# 00 — 비전 (Vision)

## 한 줄 정의

> 일기를 쓸 때마다 화분에 물이 주어지고 식물이 자라는 감성 다이어리 Flutter 앱

## 배경 & 동기

현대인은 일기를 쓰고 싶지만 꾸준히 유지하기 어렵습니다.
단순한 텍스트 메모 앱 대신, **식물 키우기**라는 시각적 보상을 통해
사용자가 기록 습관을 자연스럽게 유지하도록 동기 부여합니다.

## 목표 사용자

- 꾸준한 기록 습관을 만들고 싶은 사람
- 복잡한 앱보다 심플하고 예쁜 앱을 선호하는 사람

## 핵심 가치 제안

| 기존 일기 앱 | 나의 숲 |
|------------|--------|
| 기록해도 변하는 것 없음 | 기록하면 식물이 자람 |
| 반복적이고 지루한 UI | 매일 달라지는 화분 화면 |
| 기록 유지 동기 없음 | 성장하는 식물이 동기 부여 |

## 핵심 사용자 시나리오 3가지

### 시나리오 1 — 첫 일기 작성
> **페르소나**: 대학생 지은 (22세), 다이어리 앱을 처음 쓰는 사용자

1. 앱을 처음 실행하면 씨앗 🌰 화면이 보인다
2. "+ 일기 쓰기" 버튼을 눌러 오늘의 감정을 짧게 적는다
3. 저장하자마자 물방울 파티클이 화면에 쏟아지고 씨앗이 새싹 🌱으로 바뀐다
4. **결과**: "내가 한 행동이 즉시 결과로 보인다" → 다음 날도 앱을 켜게 된다

### 시나리오 2 — 꾸준한 기록으로 성장 확인
> **페르소나**: 직장인 민준 (28세), 한 달째 매일 일기를 쓰는 사용자

1. 홈 화면을 켜면 이미 나무 🌳 단계의 식물이 화분 안에 크게 자라 있다
2. 성장도 바에서 숲까지 앞으로 몇 개 더 남았는지 확인한다
3. 오늘 일기를 쓰고 저장하면 나무가 살짝 더 커진 느낌을 받는다
4. **결과**: "꾸준함이 눈에 보인다" → 기록을 멈추고 싶지 않아진다

### 시나리오 3 — 과거 기록 돌아보기
> **페르소나**: 고등학생 수아 (17세), 감정 정리를 위해 앱을 쓰는 사용자

1. 홈 화면 하단의 "기록 목록" 버튼을 누른다
2. 날짜 역순으로 정렬된 과거 일기 목록을 스크롤한다
3. 한 달 전에 쓴 일기를 눌러 그때의 감정을 다시 읽는다
4. **결과**: "내가 이렇게 많이 썼구나" → 성취감과 함께 지속 동기 강화

## 성공 기준

- 앱 실행 시 현재 식물 단계가 즉시 보임
- 일기 작성 → 저장 → 파티클 → 식물 성장 흐름이 2초 이내 완료
- 20개 일기 작성 시 최종 단계(숲)에 도달
