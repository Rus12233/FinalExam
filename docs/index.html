왜 우리는

😔 문제 정의

현대인은 감정을 기록하고 싶지만, 일반 메모앱은 피드백이 없어 지속 동기가 없다.
꾸준히 써야 한다는 압박만 있고, 돌아올 보상이 보이지 않는다.

✍️

일기 작성

제목 + 본문으로 감정을 자유롭게 기록

🌱

식물 성장

글을 쓸수록 씨앗 → 새싹 → 어린나무 → 나무 → 숲으로 성장

🎞️

물방울 파티클

저장 시 애니메이션으로 즉각적인 시각 보상

📅

기록 목록

과거 일기 날짜순 조회, 나의 숲 히스토리 확인

"매일 한 줄의 기록이
언젠가 울창한 나만의 숲이 된다."

Work Breakdown Structure

프로젝트 WBS

# 작업 항목 담당 상태 진행률
📐 Phase 1 — 기획 & 설계
1.1 요구사항 정의 및 기획서 작성 전체 완료
1.2 앱 아키텍처 설계 (Provider 패턴) 전체 완료
1.3 데이터 모델 설계 (Hive Box) 전체 완료
1.4 UI/UX 와이어프레임 전체 완료
🛠️ Phase 2 — 핵심 기능 개발
2.1 일기 작성 화면 (제목 + 본문) 전체 완료
2.2 Hive 로컬 저장 / 불러오기 전체 완료
2.3 식물 성장 단계 로직 (일기 수 기반) 전체 완료
2.4 홈 화면 — 씨앗/새싹/어린나무/나무/숲 표시 전체 완료
2.5 기록 목록 화면 (날짜순 조회) 전체 완료
✨ Phase 3 — 애니메이션 & 감성 UX
3.1 물방울 파티클 애니메이션 (저장 시) 전체 완료
3.2 식물 성장 전환 애니메이션 전체 진행중
3.3 감성 색상 테마 (단계별 배경색) 전체 완료
🧪 Phase 4 — 테스트 & 마무리
4.1 단위 테스트 (Provider / Hive) 전체 진행중
4.2 Android / iOS 디바이스 테스트 전체 예정
4.3 README 및 GitHub Pages 정비 전체 진행중
4.4 최종 발표 준비 전체 예정
