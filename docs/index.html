기획 · 비전 · 문제 정의

왜 우리는
일기를 지속하지 못할까?

😔 문제 정의

현대인은 감정을 기록하고 싶지만, 일반 메모앱은 피드백이 없어 지속 동기가 없다.
꾸준히 써야 한다는 압박만 있고, 돌아올 보상이 보이지 않는다.

✍️

일기 작성

기분 선택(4종) + 본문으로 감정을 자유롭게 기록

🌱

식물 성장

글을 쓸수록 씨앗 → 새싹 → 어린나무 → 나무 → 숲으로 성장

💧

물방울 파티클

저장 시 애니메이션으로 즉각적인 시각 보상 제공

📅

달력 조회

기분별 컬러 dot + 날짜 탭으로 빠른 작성/조회

🔥

연속 기록

매일 기록하면 streak이 올라가며 지속 동기 부여

🏅

업적 시스템

기록 수·연속일 달성 시 뱃지 잠금 해제

"매일 한 줄의 기록이
언젠가 울창한 나만의 숲이 된다."

구현된 화면 · 기능

4개 화면
전체 구현 완료

🔐

로그인 화면

  • 이메일·비밀번호 로그인 / 회원가입 (탭 전환)
  • 닉네임 설정 (회원가입 시)
  • 게스트로 시작하기 (로그인 불필요)
  • 입력 유효성 검사 + 에러 배너
  • 로그인 상태 Hive 영구 저장
🏠

홈 화면

  • 현재 식물 단계 시각화 (PlantWidget)
  • 성장 진행바 + 다음 단계 힌트
  • 이달 기록 수 / 총 물방울 통계 카드
  • 💧 물방울 파티클 애니메이션 (저장 시)
  • 월 네비게이션 (이전/다음 달)
  • ✍️ 홈에서 바로 일기 쓰기 — FAB 버튼으로 즉시 작성
  • 오늘 기록 없을 때 작성 유도 카드 표시
  • 오늘 기록 완료 시 건수 확인 + 추가 작성 지원
📅

달력 화면

  • 월별 달력 그리드 (일요일 기준)
  • 기분별 컬러 dot 표시
  • 날짜 탭 → 하단 시트에서 일기 조회
  • 빠른 작성 시트 (기분 + 본문)
  • 소급 작성 지원 (과거 날짜 기록)
  • 이달 기분 분포 통계
👤

프로필 화면

  • 닉네임 표시 + 수정 (최대 15자)
  • 로그인 시 이메일 표시 / 게스트 구분
  • 통계: 총 기록 / 연속 기록 / 총 물방울
  • 식물 성장 여정 (5단계 달성 현황)
  • 감정 분포 차트
  • 업적 시스템 6종
  • 설정: 닉네임 변경 / 전체 기록 삭제 / 로그아웃

📅 Phase 3 — 달력 & 프로필
3.1달력 화면 — 월별 그리드 + 기분 dot완료

3.4프로필 화면 — 닉네임 + 통계 3종완료
3.5연속 기록 (streak) 계산완료
3.6업적 시스템 6종완료

🔐 Phase 4 — 인증 시스템
4.1AuthProvider — 로그인·회원가입·게스트 모드완료
